📚 Course Recommendation System

A simple Machine Learning-based Course Recommendation System built using Streamlit and Random Forest Classifier.
It recommends the most relevant subjects based on user inputs like Branch, College, Course, and Year.


🚀 Features-
🎯 Predicts top subjects using ML
📊 Uses Random Forest for accurate recommendations
🧠 Confidence score for each recommended subject
⚡ Interactive UI with Streamlit


🛠️ Tech Stack--
Python
Streamlit
Pandas, NumPy
Scikit-learn

How to run

git clone https://github.com/your-username/repo-name.git
cd repo-name
pip install -r requirements.txt
streamlit run app.py
